Website Portfolio

Coded-Tees: Programming Tee Shirts

Coded-Tees was created while studying Internet Dynamic Environment Design. The course goal was to use the WordPress and build a Content Management System (CMS) to create a dynamically populated eCommerce website. Plugins provided the following relevant enhancements: eCommerce pages, search bar, picture gallery, and improved security.

Personal Goal

My personal goal was to create a site that was also responsive for medium to large screen sizes. The experience gained is a springboard for leaping into responsive design ranging from mobile to desktop.

The Hobbit: The Desolation of Smaug

The Hobbit site was created while studying Web Programming. The course goal was to introduce Web protocols: HTML5, CSS, SSI and XML, client side processing with JavaScript and server side processing with PHP.

The site's main component was an eCommerce shop with a shopping cart that required user accounts. This component involved using sessions and validation in JavaScript and PHP.

Personal Goal

My personal goal was create a fluid layout website with a tiny size entirely from scratch. I wanted to get back to basics, as several jQuery plugins were required for the multimedia component of the previous site; Eungella Platypus. I was pleased with the final result of a site under 3MB with fluid media.

Eungella National Park Platypus & Friends

Eungella National Park Platypus was created while studying Internet Interactivity Design . The course goal was to introduce interactivity design through creating an interactive site using animation and other multimedia approaches. The jQuery plugins provided the following functionality: tour of the site's interactivity, video player, picture slideshow, and comment section.

Personal Goal

My personal goal was to experiment with advanced features of each jQuery plugin to create a well interactive site.

Photoshop Logo Portfolio

KEPT: Keep Educating Parents Together

An existing client approached me with a clear idea of the logo. I traced a photograph of posed hands using the Photoshop pen tool, then from that base I abstracted the design. The Merienda One font was sourced from Google Fonts and falls under the SIL Open Font License, 1.1. The KEPT logo is intended to be printed onto Tee Shirts for the cause of Education Reform in the United States.

Personal Goal

My personal goal was to accurately convey the client's visualisation of the logo. The requirements involved an abstract drawing of an adult and a child with interlocking little fingers; a pinky promise.

NightStar Images Photography

I designed this logo for a client's photography business. It serves as both the business logo and the watermark used on images. The client chose the font as it was inspired by Film Producer Tim Burton's dark and artistic style. I added the shooting star and used the Photoshop pen tool to create an ideal trail arc.

Personal Goal

My personal goal was to test requirements elicitation theory that I've learned in software development courses. Through rapid prototyping, the logo was designed over ten iterations each with client feedback.
